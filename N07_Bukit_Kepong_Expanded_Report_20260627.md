## **2.2 Verification Methodology — Detailed Protocol**

### **2.2.1 Multi-Source Corroboration Rule**

**Standard:** All critical claims (candidate confirmation, election results, demographic data) require **minimum 2 independent Tier 1 sources** OR **1 Tier 1 + 2 Tier 2 sources**.

| Claim Type | Required Corroboration | Example |
|------------|------------------------|---------|
| **Candidate Nomination** | 2x T1 (Party announcement + SPR/Wikipedia) | Ahmad Syarie: UMNO statement + Malaysia Tribune |
| **Election Results** | 2x T1 (SPR + Tindak Malaysia) | 2022 results: Wikipedia + Tindak sheet |
| **Demographic Data** | 1x T1 + 1x T2 (DOSM + academic study) | 2020 Census + university demographic analysis |
| **Policy Announcements** | 2x T1 (Official statement + mainstream media) | FELDA debt relief: FELDA.gov.my + The Star |
| **Scandal/Controversy** | 3x T1/T2 (Media + official response + documentation) | Party-hopping: 3 media outlets + parliamentary record |

### **2.2.2 Temporal Validity Assessment**

| Data Type | Validity Window | Flag if Older Than | Action Required |
|-----------|-----------------|--------------------|-----------------|
| **Candidate Status** | 7 days | 14 days | Re-verify via party announcement |
| **Polling Data** | 30 days | 60 days | Note as outdated; seek newer poll |
| **Demographic Data** | 2 years | 5 years | Use with caveat; note census year |
| **Economic Data** | 6 months | 12 months | Update with latest quarterly report |
| **Historical Results** | Permanent | N/A | No update needed (historical fact) |
| **Coalition Agreements** | 30 days | 60 days | Re-verify via coalition statement |

### **2.2.3 Contradiction Resolution Protocol**

When sources contradict:

1. **Step 1:** Identify source tiers (T1 > T2 > T3)
2. **Step 2:** Check publication dates (newer > older)
3. **Step 3:** Assess potential bias (non-partisan > partisan)
4. **Step 4:** Seek additional corroboration (third source)
5. **Step 5:** Document contradiction in report (transparency)
6. **Step 6:** Apply confidence labeling (HIGH/MODERATE/LOW)

**Example Application:**
- **Claim:** "PH will contest Bukit Kepong"
- **Source A:** Facebook rumor (T3, 25 June)
- **Source B:** The Star report citing PH insider (T1, 26 June)
- **Source C:** PH official statement "no decision yet" (T1, 27 June)
- **Resolution:** Source C (most recent T1) prevails → Status: PENDING
