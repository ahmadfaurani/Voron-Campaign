## 📧 Email Format Pattern

Based on verified PDRM email addresses:

**Standard Format:**
- `bp_dcc[district]@rmp.gov.my` (District Police)
- `cccmel@rmp.gov.my` (Contingent Communication Centre)
- `cpo.melaka@rmp.gov.my` (Chief Police Officer)
- `rmp@rmp.gov.my` (HQ General)
- `pjpswk@rmp.gov.my` (Sarawak CPO)

**Email Pattern Notes:**
- District codes: `bp_dcc` + district abbreviation
- State CPO: `cpo.[state]@rmp.gov.my`
- Communication centres: `ccc[state]@rmp.gov.my`

## 📋 Data Collection Notes

### Source Hierarchy (Confidence Level)
1. **Highest:** Official PDRM portal (rmp.gov.my) - direct district pages
2. **High:** MERS 999 government portal (999.gov.my)
3. **Medium:** State/municipal government portals (e.g., mphtj.gov.my)
4. **Reference:** Third-party directories (malaysiacentral.com) - verify against official sources

### Contact Verification Protocol
- ✅ **Verified:** Direct extraction from rmp.gov.my official directory
- ⚠️ **Cross-reference:** Multiple sources match (MPHTJ, MERS 999)
- 🔄 **Needs verification:** Email addresses with spacing errors in source documents

### Known Data Quality Issues
- Some Excel/PDF exports contain spacing errors in emails (e.g., `rm p.gov.my`)
- Phone number formats vary: `06-2842222` vs `06-284 2222` vs `+606-2842222`
- Standardize to: `0X-XXXXXXX` format for domestic, `+60X-XXXXXXX` for international

## 📝 Usage Notes for Journalist Registry

### When to Use PDRM Contacts
1. **Media Relations:** Contact IPK Communication Centres (CCC) for official statements
2. **Investigation Inquiries:** District IO (Investigating Officer) contacts from press releases
3. **Event Coverage:** IPK Public Relations units for press briefings
4. **Verification:** Cross-check journalist claims about police statements

### TLP:AMBER Handling
- Contact details are publicly available from official sources
- Do not redistribute individual officer mobile numbers beyond official directory listings
- Generic newsroom/CCC emails preferred for initial media inquiries
- Individual IO contacts only when explicitly published in press statements

### Integration with Registry
- Link PDRM district contacts to corresponding outlet coverage areas
- Track which journalists have established relationships with specific IOs
- Monitor press release patterns by district for beat mapping
