## COLLECTION WORKFLOW (Per Institution)

### Step 1: Official Website Extraction (15 min)
```
1. Find leadership page: site:{domain} "leadership" OR "senior management"
2. web_extract leadership page URL
3. Extract: CEO, CFO, CRO, CIO, CISO, Compliance, Audit
4. Score: HIGH confidence (official source + Malaysia)
```

### Step 2: LinkedIn Enrichment (10 min)
```
1. Search: site:linkedin.com/in "{Institution}" "{Role}"
2. Validate: Role match + company match + Malaysia location
3. Score: MEDIUM-HIGH confidence (LinkedIn + cross-reference)
4. Add to database with source URL
```

### Step 3: Annual Report Verification (10 min)
```
1. Find annual report: site:{domain} "annual report 2024" OR "annual report 2025"
2. web_extract PDF (if supported) or extract landing page
3. Verify C-suite names against website data
4. Update confidence scores if confirmed
```

### Step 4: Database Integration (5 min)
```
1. Update master CSV with all 7 roles
2. Add source URLs for audit trail
3. Version control: prospect-database-enriched-v{version}.csv
4. Push to GitHub (Voron-Campaign repo)
```
